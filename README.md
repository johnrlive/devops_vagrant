# Vagrantfile with my favorite DevOps tools

# Batteries Included
- Ubuntu 16.04
- Ansible
- AWS CLI
- Docker
- EB CLI
- Packer
- Terraform

### Vagrant Commands 

* run Virtual machine (VM)

```
vagrant up
```

* login to VM

```
vagrant ssh
```

* destroy VM

```
vagrant destroy
```

* Halt

```
vagrant halt
```

### Hostname
* fix hostname in `/etc/hosts`
